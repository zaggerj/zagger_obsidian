# nvm

# nvm安装（ubuntu22.04）

```js
sudo apt install curl 
curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash 
source ~/.bashrc   
nvm -v
```

# nvm使用

```js
nvm install 16.14.1
nvm use 16.14.1
node -v 
nvm alias default 16.14.1
```
