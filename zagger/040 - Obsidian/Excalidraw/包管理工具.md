### 改变npm默认路径

- 步骤如下：
进入终端，依次输入一下命令
（1）创建global安装任务的目录
mkdir ~/.npm-global
（2）配置npm使用新的目录
npm config set prefix '~/.npm-global'
（3）在~/.profile文件中增加配置
export PATH=~/.npm-global/bin:$PATH
（4）配置文件立即生效
source ~/.profile
（5）重新执行命令
npm install -g xxxx 

### 成功有效的一种方式

- prefix=/mnt/d/software/Nodejs/node_global
cache=/mnt/d/software/Nodejs/node_cache
registry=http://registry.npm.taobao.org
